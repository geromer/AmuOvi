## Publishing to GitHub Pages

1. Go to your repository **Settings → Pages**
2. Under **Source**, select `Deploy from a branch`
3. Choose branch `main`, folder `/ (root)`
4. Click **Save** — the site is live at `https://gerom.github.io/AmuOvi/` within a minute or two

## Adding a new guide

1. Create a new `.html` file in the repo root (follow the style of the existing guides)
2. Add a card for it in `index.html`
3. Add a row to the table above in this README
